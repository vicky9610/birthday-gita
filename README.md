# Premium Birthday Surprise Website

Welcome to your cinematic single-page birthday experience!

## How to use this project

To personalize this website, you only need to drop your files into the appropriate folders. The code will automatically load them.

1. **Photos (`/photos/`)**: 
   - Add exactly 60 photos into this folder.
   - Name them `photo1.jpg`, `photo2.jpg`, up to `photo60.jpg`.
   - The Photo Memory section will automatically detect and load them.

2. **Music (`/music/`)**:
   - Add your background music file here.
   - Name it `bg-music.mp3`.

3. **Videos (`/videos/`)**:
   - If you have a special video, place it here and name it `memory.mp4`.

## Running the Website

Because this project uses advanced JavaScript and modules, it is best run through a local web server (like the VS Code Live Server extension). Simply open `index.html` with Live Server.

Enjoy! ❤️
